## Описание проекта

В данном репозитории представлены решения задач контрольной работы №2 по созданию серверных приложений на базе фреймворка FastAPI. [cite_start]Проект разделен на несколько логических модулей согласно выданным заданиям[cite: 9].

## Структура репозитория

* [cite_start]`task_3_1.py` — **Задание 3.1**: Конечная точка `POST /create_user` для регистрации пользователей с валидацией данных через Pydantic (проверка email и возраста)[cite: 10, 11, 14, 15].
* `task_3_2.py` — **Задание 3.2**: Маршруты для работы с товарами (`GET /product/{product_id}` и `GET /products/search`). [cite_start]Обработка конфликтов маршрутов[cite: 35, 38, 45, 108].
* `task_5.py` — **Задания 5.1 - 5.3**: Реализация безопасной аутентификации на основе файлов cookie. [cite_start]Включает генерацию HMAC-подписей и механизм динамического продления времени жизни сессии (максимум 5 минут, обновление после 3 минут)[cite: 110, 111, 133, 161, 166].
* [cite_start]`task_5_4.py` — **Задание 5.4**: Извлечение HTTP-заголовков `User-Agent` и `Accept-Language` напрямую из запроса[cite: 240, 243, 244].
* `task_5_5.py` — **Задание 5.5**: Оптимизация работы с заголовками (принцип DRY) с использованием модели Pydantic `CommonHeaders` и FastAPI `Depends`. [cite_start]Добавление кастомного заголовка ответа `X-Server-Time`[cite: 266, 267, 274, 279].
* `requirements.txt` — Список зависимостей для запуска проекта.

## Установка и запуск

1.  Установите необходимые зависимости:
    ```bash
    pip install -r requirements.txt
    ```
2.  Для запуска конкретного задания используйте сервер Uvicorn. Например, для запуска заданий 5.1-5.3:
    ```bash
    uvicorn task_5:app --reload
    ```
3.  Интерактивная документация (Swagger UI) будет доступна по адресу: `http://127.0.0.1:8000/docs`.
